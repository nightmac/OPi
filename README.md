# OPi5 setup script

### Installs or configures

- KStars
- INDI
- INDIGO
- INDIGO Control Panel
- AIN Imager
- PHD2
- Siril
- x11vnc
- SMB file sharing
- Avahi
- iPhone USB Tethering
- iPerf3
- Synaptic package mangaer
- Flatpak 
- zshrc & inputrc dotfiles
- Python3
- Pyindi-client
- other misc things


### Installing

`cd ~/Desktop`

`git clone https://github.com/nightmac/OPi.git && cd OPi`

`chmod +x OPi-setup.sh`

`sudo ./OPi-setup.sh`
