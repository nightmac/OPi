# OPi5 setup script

### Installs or configures

- KStars
- INDI
- Indigo
- AIN Imager
- Indigo Control Panel
- PHD2
- Siril
- X11vnc
- SMB file sharing
- Avahi
- iPhone USB Tethering
- iPerf3
- Synaptic package mangaer
- zshrc & inputrc dotfiles
- Python3
- Pyindi-client
- other misc things


### Installing

`cd ~/Desktop`

`git clone https://github.com/nightmac/OPi.git`

`cd OPi`

`chmod +x OPi-setup.sh`

`./OPi-setup.sh`
